# Security Policy

## Reporting a vulnerability

Please do not publish exploit details, credentials, tokens, private configuration, or user data in a public issue.

Open a GitHub Security Advisory for the repository when available. Include the affected patcher version, Codex Desktop version, reproduction steps, and expected impact.

## Scope

This project only supports its own scripts and launcher code. It does not provide security support for OpenAI, Codex Desktop, Microsoft Store, third-party model providers, or user accounts.

## Distribution safety

Official releases publish a ZIP and a separate SHA-256 file. The repository never distributes Codex Desktop binaries, account data, model catalogs, or API credentials.
